Notes:
  • Areas use π·D²/4 (mm²). Flow L/min = area(m²) × speed(m/min) × 1000.
  • Force kN = (Area mm² × Pressure(bar) × 0.1) / 1000.
  • Pressure/Force input is bidirectional — entering one computes the other.
  • Regeneration (default off) affects only the power calculation: if enabled, power uses bore flow (Qb). If disabled, power uses (Qb − Qa).
